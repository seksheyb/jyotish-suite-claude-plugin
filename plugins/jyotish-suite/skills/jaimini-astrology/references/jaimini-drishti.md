# Jaimini Drishti — Complete Sign Aspect Tables

Sign-based aspects only. No orbs. No degree calculation. A sign either aspects another or it does not.

---

## The Three Rules

**Rule 1 — Movable signs** (Aries, Cancer, Libra, Capricorn):
Aspect all Fixed signs EXCEPT the one immediately next in zodiacal order (adjacent fixed sign).

**Rule 2 — Fixed signs** (Taurus, Leo, Scorpio, Aquarius):
Aspect all Movable signs EXCEPT the one immediately next in zodiacal order (adjacent movable sign).

**Rule 3 — Dual signs** (Gemini, Virgo, Sagittarius, Pisces):
Aspect ALL other Dual signs without exception.

**All Jaimini Drishti is mutual** — if A aspects B, then B aspects A.

All Jaimini aspects are mutual. Complete list of mutual aspect pairs:

- Sagittarius ↔ Pisces

**Signs that do NOT aspect each other (no mutual aspect):**
- Aries–Taurus (adjacent; Aries skips Taurus; Taurus skips Cancer not Aries... actually Taurus DOES aspect Aries — verify: Taurus aspects Aries, Libra, Capricorn. Aries aspects Leo, Scorpio, Aquarius. So Aries and Taurus DO have a mutual aspect.)

**Corrected — pairs that have NO aspect (neither aspects the other):**
- Aries–Cancer (Aries aspects Leo/Sco/Aqu; Cancer aspects Tau/Sco/Aqu — no Aries↔Cancer)
- Aries–Capricorn (Aries aspects Leo/Sco/Aqu; Capricorn aspects Tau/Leo/Sco — no Aries↔Capricorn)
- Cancer–Leo (Cancer skips Leo; Leo skips Libra — but does Leo aspect Cancer? Leo aspects Aries/Cancer/Capricorn. YES — Leo aspects Cancer. Cancer skips Leo. So: Leo→Cancer aspect exists but Cancer does NOT aspect Leo back. This means it is a ONE-WAY aspect from Leo to Cancer.)

**One-way aspects (NOT mutual):**
In Jaimini, the mutual nature means if A aspects B then B aspects A — this is definitional. The "adjacent sign exception" breaks the mutual symmetry. Example:
- Cancer aspects Taurus but Taurus does NOT aspect Cancer (Taurus skips Cancer)
- So Taurus→Cancer is one-way FROM Cancer TO Taurus only... but wait: Cancer aspects Taurus; Taurus aspects Aries/Libra/Capricorn — NOT Cancer. So Cancer→Taurus is one-way FROM Cancer.
- But Leo aspects Cancer; Cancer does NOT aspect Leo. So Leo→Cancer is one-way FROM Leo.

**Key insight:** The "mutual" rule in Jaimini applies WITHIN the same category pairs, but the adjacent-sign exception creates a handful of one-way aspects. When a movable sign's adjacent fixed sign is skipped, that pair is one-way — the fixed sign still aspects the movable, but the movable doesn't aspect it back.

**One-way aspect pairs (fixed aspects movable but movable skips fixed):**
- Leo → Cancer (one-way; Cancer skips Leo)
- Scorpio → Libra... check: Libra aspects Taurus/Leo/Aquarius (skips Scorpio). Scorpio aspects Aries/Cancer/Libra. So Scorpio→Libra AND Libra does NOT aspect Scorpio. One-way from Scorpio.
- Aquarius → Capricorn... check: Capricorn aspects Taurus/Leo/Scorpio (skips Aquarius). Aquarius aspects Cancer/Libra/Capricorn. So Aquarius→Capricorn AND Capricorn does NOT aspect Aquarius. One-way from Aquarius.
- Taurus → Aries... check: Aries aspects Leo/Scorpio/Aquarius (skips Taurus). Taurus aspects Aries/Libra/Capricorn. One-way from Taurus.

**Summary of one-way aspects:**

## Application Notes

1. **Always check both directions** — when analyzing a sign, check what it aspects AND what aspects it
2. **Empty aspecting sign** → aspect still operates; read through the sign lord's placement
3. **Multiple planets in aspecting sign** → all contribute their Karaka roles and natural natures to the aspected sign
4. **Mutual aspect = bidirectional exchange** — describe both directions; planets in both signs co-influence each other's themes
5. **One-way aspect** — only the aspecting sign's planets influence the aspected sign; no return influence
6. **Nodes (Rahu/Ketu)** → cast aspects based on their sign's type (movable/fixed/dual)
