# Navamsa (D9) Computation Table

Each sign is divided into 9 Navamsas of 3°20' each.
The starting Navamsa sign depends on the element of the D1 sign.

## Starting Points by Element

| D1 Sign Element | First Navamsa Starts In |
|----------------|------------------------|
| Fire (Aries, Leo, Sagittarius) | Aries |
| Earth (Taurus, Virgo, Capricorn) | Capricorn |
| Air (Gemini, Libra, Aquarius) | Libra |
| Water (Cancer, Scorpio, Pisces) | Cancer |

## Vargottama Check

A planet is Vargottama if its D1 sign = its D9 sign.

From the table above, this happens when a planet is in the same Navamsa position as the sign itself:
- Aries: 0°00'–3°19' (Navamsa = Aries)
- Taurus: 3°20'–6°39' in Taurus (Navamsa = Taurus)
... and so on — when the Navamsa sign cycles back to match the D1 sign.

Vargottama planets are considered greatly strengthened — they carry their D1 promise into the soul level (D9) without dilution.

## Navamsa Lagna Calculation

The Navamsa Lagna is computed the same way as any planet — apply the Navamsa table to the D1 Lagna degree. The resulting sign becomes the Navamsa Ascendant, and houses in D9 are counted from there using Whole Sign Houses.
